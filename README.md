🛡️ Cybersecurity Hands-On Roadmap — Do Zero ao Threat Hunter

Um roadmap prático de cibersegurança focado em aprendizado real através de laboratórios.

Este projeto foi criado para estudar segurança ofensiva e defensiva simulando atividades reais de um analista SOC / Blue Team.

🎯 Objetivo

Aprender cibersegurança da forma mais próxima possível do mundo real:

Construindo um ambiente isolado

Gerando ataques controlados

Investigando logs e tráfego

Criando detecções

Realizando threat hunting

Tudo utilizando ferramentas gratuitas ou amplamente usadas no mercado.
